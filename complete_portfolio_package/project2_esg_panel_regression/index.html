ESG Scores and Corporate Financial Performance

Panel data regression analysis testing whether higher ESG scores predict better financial outcomes — using fixed effects, random effects, and the Hausman test

🐍 Python & Stata📊 Panel FE/RE Regression📅 200 firms × 10 years📈 ROA & Tobin's Q

1. Research Question & Motivation

"Do higher ESG scores predict better financial performance (ROA, Tobin's Q) after controlling for firm-level unobserved heterogeneity?"

The relationship between ESG and financial performance is one of the most debated questions in sustainable finance. Simple cross-sectional regressions are plagued by omitted variable bias — profitable firms may invest more in ESG, creating reverse causality. Panel data with firm fixed effects addresses this by controlling for all time-invariant firm characteristics (management quality, culture, brand strength) that might confound the ESG–performance relationship.

2. Data Collection & Sources

Structure
Balanced panel: 200 firms × 10 years = 2,000 firm-year observations
Dependent Variables
ROA (Return on Assets), Tobin's Q (market value / book value)
Key Independent Variable
ESG composite score (0–100) plus E, S, G pillar scores
Controls
Firm size (log assets), leverage (debt/equity), R&D intensity, sector dummies
Source
Simulated data calibrated to match Refinitiv/Compustat distributional properties
import pandas as pd
import numpy as np

# Load panel dataset
panel = pd.read_csv('esg_panel_data.csv')
panel = panel.set_index(['firm_id', 'year'])

# Summary statistics
print(panel[['roa', 'tobins_q', 'esg_score', 'size', 'leverage']].describe())

3a. Pooled OLS (Baseline)

Ignores panel structure — treats all observations as independent. Provides a naive benchmark but is biased if unobserved firm characteristics correlate with ESG scores.

import statsmodels.formula.api as smf

# Pooled OLS (biased baseline)
pooled = smf.ols('roa ~ esg_score + size + leverage + C(sector)', data=panel).fit()
print(f"Pooled OLS — ESG coefficient: {pooled.params['esg_score']:.4f}")

4a. Fixed Effects (Within Estimator)

Controls for ALL time-invariant firm characteristics by demeaning each variable within firms. The gold standard for causal inference in panel data when unobserved heterogeneity is a concern.

# Stata: Fixed Effects with clustered standard errors
* xtset firm_id year
* xtreg roa esg_score size leverage, fe cluster(firm_id)

# Python equivalent
from linearmodels.panel import PanelOLS
fe_model = PanelOLS.from_formula(
    'roa ~ esg_score + size + leverage + EntityEffects',
    data=panel
).fit(cov_type='clustered', cluster_entity=True)

5a. Hausman Test

Tests whether the random effects estimator is consistent. If the test rejects (p < 0.05), firm-specific effects are correlated with regressors, and fixed effects is preferred.

# Stata Hausman test
* xtreg roa esg_score size leverage, fe
* estimates store fe
* xtreg roa esg_score size leverage, re
* estimates store re
* hausman fe re

Key Findings

1
Hausman test strongly rejects random effects (p < 0.01), confirming that unobserved firm heterogeneity is correlated with ESG scores — fixed effects is the appropriate estimator
2
Under fixed effects, ESG composite shows a positive but modest association with ROA (β ≈ 0.03, p < 0.05) — within-firm ESG improvements predict slightly higher profitability
3
The Governance pillar has the strongest individual relationship with Tobin's Q, while the Environmental pillar shows the weakest — suggesting markets primarily value governance improvements
⚠️ Methodological Caveat

Panel fixed effects only control for time-invariant confounders. Time-varying omitted variables (e.g., a new CEO who simultaneously improves ESG and profitability) can still bias estimates. Instrumental variables or dynamic panel methods (Arellano-Bond) are needed for stronger causal claims.
