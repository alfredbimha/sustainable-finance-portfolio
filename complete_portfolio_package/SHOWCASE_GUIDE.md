## 5. INTERVIEW TALKING POINTS

When asked "Tell me about a project you've worked on":

"I built a 20-project research portfolio in sustainable finance. For example, one
project analyses SEC 10-K filings using NLP — I found that energy companies mention
climate-related terms about 39 times per filing while tech companies mention them
almost zero times. Another project uses cointegration analysis on real Yahoo Finance
data and finds that clean and fossil energy ETFs share a long-run equilibrium with
a p-value of 0.03."

When asked "How comfortable are you with Python/R/Stata?":

"I use all three across my portfolio. Python is my primary tool for data collection
and analysis — I pull data from Yahoo Finance, SEC EDGAR, and the World Bank API.
I use R for structural topic modelling and visualisation, and Stata for panel
econometrics where it's the industry standard. Each project has full code with
detailed comments explaining the methodology."

When asked about AI/tools:

"I use AI as a coding collaborator, which I think is an important modern skill.
I direct all the research — choosing the questions, selecting methods, interpreting
results — while using Claude to help implement the code. This lets me focus on the
analytical thinking while maintaining transparency about the workflow. I think this
is how data-intensive research will increasingly be done."
